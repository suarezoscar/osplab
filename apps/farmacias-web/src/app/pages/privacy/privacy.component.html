Política de Privacidad

Última actualización: {{ updated }}

1 Responsable

El responsable de este sitio web es Oscar Suarez. Puedes contactar a través de GitHub o LinkedIn.

2 Datos que se tratan

  • Geolocalización — solo se usa en el momento de la consulta para calcular las farmacias más cercanas. No se almacena, no se registra y no se comparte.
  • Búsquedas por dirección — el texto introducido se envía a la API de geocodificación de Nominatim / OpenStreetMap para resolver coordenadas. No se guarda ningún historial.
  • No se usan cookies de seguimiento ni analítica de terceros. No hay registro de usuarios ni inicio de sesión.

3 Origen de los datos de farmacias

La información sobre farmacias de guardia (nombre, dirección, horario y teléfono) se obtiene de fuentes públicas oficiales: los Colegios Oficiales de Farmacéuticos de Galicia (COF A Coruña, COF Lugo, COF Ourense, COF Pontevedra). Los datos se actualizan diariamente de forma automática. Si detectas algún error, puedes comunicarlo a través de GitHub.

4 Tus derechos

Dado que no se recopilan ni almacenan datos personales de los usuarios, no aplican derechos de acceso, rectificación o supresión sobre información propia. Si tienes alguna duda, escríbeme a través de los canales indicados en el punto 1.

5 Cambios en esta política

Cualquier modificación relevante se reflejará en la fecha de actualización indicada al inicio de este documento.
